Icon-only square control for social links, nav, and toolbars. Always provide `label`.

```jsx
<IconButton label="GitHub" href="#"><i data-lucide="github"></i></IconButton>
<IconButton label="Close" tone="ink" bare><i data-lucide="x"></i></IconButton>
```

44px hit target by default. `tone="ink"` for dark sections; `bare` removes the border. Hover turns the icon and border amber.
