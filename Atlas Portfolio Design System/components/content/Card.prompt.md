Surface container for project & certification grids and feature panels. `hover` for clickable lift; `icon` renders an amber icon chip at top; `tone="ink"` on dark sections; `flush` removes padding for media.

```jsx
<Card hover icon={<i data-lucide="layers"></i>}>
  <Badge variant="accent">Platform</Badge>
  <h3>Payments migration</h3>
  <p>Re-platformed checkout to a new ledger with zero downtime.</p>
</Card>
```
